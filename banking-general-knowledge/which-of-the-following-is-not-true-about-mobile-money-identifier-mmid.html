EXPLANATIONS BELOW

Concept note-1: -MMID (Mobile Money Identifier) is a 7-digit code that is issued by a bank to its customer in order to avail the mobile banking services. Each mobile banking account has a unique MMID allotted to the user by the bank itself.

Concept note-2: -MMID is an acronym for mobile money identifier. It is a seven-digit unique code assigned to a bank account. It was introduced as a successor to standard money transfer methods like NEFT and RTGS in 2010.

Concept note-3: -Mobile Money Identifier, which is also abbreviated as MMID, is a seven digit unique number that is used for interbank transfer during mobile banking using the MMID feature. Each account holder will have an MMID and this will help in the identification of a particular bank account.
