Which statement is correct for Green PIN?

(A) Green PIN facility is available only for domestic customers

(B) Green PIN can not be generated for NRE/NRO customer

(C) Only physical pin can be generated for NRI customer using Finacle/BOB World/IVR/Baroda connectGreen PIN can not be generated for NRE/NRO customer

(D) ** all are correct

EXPLANATIONS BELOW

Concept note-1: -The main features of the Green Pin are: (i) OTP of 6 digits is sent to Registered Mobile Number (ii) It is valid only for 3 minutes from its delivery(for security reasons) (iii) It can be used only once. (iv) It will work only in IOB ATMs (v) Green Pin is not applicable for NRE Accounts.

Concept note-2: -GREEN PIN, the green initiative & a value added service, offers an effortless and hassle free ATM PIN generation for existing as well as new debit card customers. Customers can set their ATM PIN for new debit cards instantly after receiving the debit card by visiting our Bank ATM/ Other Bankâ€™s ATM.

Concept note-3: -Please send SMS DCPIN space CARDNUMBER to 5607040 from registered mobile number for active debit card. Immediately on successful validation of your card/mobile credentials, system will send 6 digits OTP on registered mobile number with validity of 72 hours.

Concept note-4: -Green PIN initiative offers a hassle free mechanism to generate Debit-cum-ATM PIN for existing as well as new Debit card customers. Customers can set their PIN for new debit cards instantly after receiving the debit card or make use of this functionality in case they forget their existing Debit-cum-ATM Card PIN.
