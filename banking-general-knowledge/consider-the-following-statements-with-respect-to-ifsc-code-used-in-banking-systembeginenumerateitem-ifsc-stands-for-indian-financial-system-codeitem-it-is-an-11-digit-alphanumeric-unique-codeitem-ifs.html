1. :

3. :

(A) Only 1

(B) Only 2

(C) Only 3

(D) ** All 1, 2, and 3

EXPLANATIONS BELOW

Concept note-1: -The IFSC code is a 11-digit alpha-numeric code needed to transfer money between accounts in India. IFSC stands for Indian Financial System Code. The Reserve Bank of India (RBI) utilises the IFSC code for electronic money transfers between banks, as it helps them verify the transfer of resources.

Concept note-2: -IFSC Code is a unique eleven-digit number which is a combination of alphabets and numbers. IFSC code is allotted by the Reserve Bank of India (RBI) to all banks and its branches.

Concept note-3: -The first 4 digits of the IFSC represent the bank and last 6 characters represent the branch. The 5th character is zero.

Concept note-4: -IFSC is an 11-digit alpha-numeric code that uniquely identifies a bank-branch participating in the RBI regulated funds transfer system. MICR is a 9 digit unique code that identifies bank and bank branches in the ECS for the transfer of funds via cheque.
