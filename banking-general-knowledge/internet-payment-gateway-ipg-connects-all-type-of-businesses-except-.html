Internet Payment Gateway (IPG) connects all type of businesses except ____

(A) B2B

(B) ** G2G

(C) C2C

(D) B2C

EXPLANATIONS BELOW

Concept note-1: -Internet Payment Gateway (IPG) is an e-commerce application service that authorizes payments for e-businesses e.g online payments (fee/fines/deposits)

Concept note-2: -IPG is a premium service for Merchants with a transaction capable website. All transactions come with 128 bit security and two factor authentication via MasterCard Secure Code and Verified-by-Visa authentication.

Concept note-3: -There are 4 types of payment gateway: Hosted payment gateway. Self-hosted payment gateway. API hosted payment gateway. Local bank integration gateway.

Concept note-4: -A payment gateway is a service that allows businesses to accept payments online. The payment gateway acts as a go-between between the business and the customer.
