Using a PIN is optional when using a debit card.

(A) True

(B) ** False

EXPLANATIONS BELOW

Concept note-1: -False. Using a PIN (Personal Identification Number) is optional when using a debit card. True/False. Check 21 allows individuals to bounce checks easily.

Concept note-2: -PIN numbers are not required to use your debit card. You can continue to use the card by signing for purchases. Purchases made using a PIN could be denied if the merchant decides not to allow healthcare purchases on the PIN network. PIN numbers cannot be changed.

Concept note-3: -Because the PIN authorizes you (or whoever knows it) to access sensitive information, it’s essential to keep the number secret. Protect it, and never write it on your ATM or debit card-thieves know to look for four-digit codes written on the back of stolen cards.

Concept note-4: -Your debit card PIN is typically a four-digit number that lets you access your account. A debit card security code (CSC) is distinct from your PIN. Sometimes called the card verification value or card verification code (CVV or CVC, respectively), it’s a three or four-digit number printed somewhere on your debit card.
